
# Speedway Surfers

**Speedway Surfers** is a 2D arcade-style game developed for the ECE319H: Intro to Embedded Systems Honors final project. Inspired by the popular game *Subway Surfers*, this project features custom hardware and optimized firmware to deliver a fun gaming experience!

## Gameplay Overview

The player controls a sprite navigating through the UT Austin campus, dodging obstacles inspired by UT.

* **Obstacles:** Avoid incoming people, scooters, and club tents.
* **Languages:** Supports both **English** and **French** language options, selectable at the start.
* **Controls:** Integrated physical buttons for jumping and ducking, with a slide potentiometer (Slide Pot) for horizontal movement.
* **Audio:** Real-time background music with a dedicated mute/unmute button.

##  How to Play

1. **Start:** Press the `START` button to initialize the game.
2. **Language:** Use the **Slide Pot** to toggle between English and French, then press `START` again to confirm.
3. **Movement:**
* **Slide Pot:** Move left and right.
* `SW3`: Jump over low obstacles.
* `SW4`: Duck under high obstacles.

4. **Audio:** Use the dedicated mute button, `SW2`, to toggle the background music.
5. **Game Over:** The game ends upon colliding with an obstacle, displaying your final score.
